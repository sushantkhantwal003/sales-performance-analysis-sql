📊 Sales Performance Analysis (MySQL Project)
🔍 Project Overview

This project analyzes sales data using MySQL to uncover insights related to revenue, growth trends, seasonality, profitability, and product performance.

🛠️ Tools Used
MySQL Workbench
SQL (Joins, Aggregations, CASE, Group By)
📈 Key Insights
Top 5 products contribute ~93% of total revenue
Sales grew by ~65% from 2021 to 2024
~51% of sales occur in last 4 months (strong seasonality)
Chocolate division achieved 135% of target, while others underperformed
Top 2 factories contribute ~96% of total sales
Most products have 60%+ profit margins
~21% products generate ~70% sales (Pareto principle)
📊 Analysis Performed
Data Cleaning (0% nulls, 0% duplicates)
Revenue Analysis
Yearly Growth Trend
Seasonal Analysis
Target vs Actual Comparison
Factory-wise Performance
Profit Margin Analysis
Product Segmentation
